# imshell

Display an image on the terminal.

## Install

```
$ go get -u github.com/rskull/imshell
```

## Usage

```
Usage of ./bin/imshell:
  -i string
        image file path
  -t string
        given text replace
  -v    imshell version
  -w float
        output width (default 50)
```

## example

```
$ imshell -i icon.png -w 30
```
